Launch Anaconda Navigator(anaconda3) on start
Launch VS Code on the dashboard of anaconda navigator
In the VS Code, open the group folder
Open a new terminal, install the following library, the second library takes more time to install
pip3 install Keras
pip3 install tensorflow
pip install flask_ngrok

Then,open app.py ,run the python file in terminal
On the terminal, there is a http address, hold ctrl key and click the link, it will direct to the chatbot interface.
